Welcome to the monorepo documentation hub.

This site collects Markdown and MDX docs from across the repository **where they live** —
no copying. Anything placed in a `docs/` directory inside an app or package is published
here automatically, grouped by package in the sidebar.

This landing page lives in the repo-root `content/` directory and is per-repo (not synced
from the white-label template). Edit it to brand your own docs home.
